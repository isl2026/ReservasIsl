# ISL - Sistema de Reservas de Salas y Notebooks

Sistema web para gestión de reservas internas de ISL Medicina Empresarial.

## Tecnologías
- React 18
- Tailwind CSS
- Supabase (base de datos)

## Instalación local
```bash
npm install
npm start
```

## Deploy
Conectar este repositorio a Netlify o Vercel para deploy automático.
